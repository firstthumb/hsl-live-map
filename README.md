# HSL Live Map

[![Build Status](https://travis-ci.org/firstthumb/hsl-live-map.svg?branch=master)](https://travis-ci.org/firstthumb/hsl-live-map)

HSL Live Map that displays trams on map

## Installation

Update `.env.development` file

```bash
yarn install
yarn start
```
